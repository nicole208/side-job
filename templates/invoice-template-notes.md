# 請求書作成テンプレートメモ

## このテンプレートの役割

実際の請求書原本を置くためではなく、請求書を作るときに最低限入れる項目と確認事項を整理するためのテンプレートです。

## 基本項目

- 請求書番号:
- 発行日:
- 請求先名:
- 差出人名:
- 件名または対象業務:
- 対象期間:
- 請求金額:
- 支払期日:
- 振込先:

## 明細欄の確認

- [ ] 何に対する請求かが分かる
- [ ] 対象期間が入っている
- [ ] 数量、単価、金額の整合が取れている
- [ ] 消費税や源泉徴収の扱いが整理できている

## 送付前チェックリスト

- [ ] 宛名が正しい
- [ ] 金額に誤りがない
- [ ] 支払期日が明記されている
- [ ] ファイル名を統一ルールで付けた
- [ ] 送付方法を確認した

## ファイル名例

- yyyy-001-invoice-client-name.pdf
- yyyy-002-invoice-client-name.pdf

- 送付手段:
- 補足:

## 注意

取引先指定フォーマットがある場合はそちらを優先します。インボイス制度や源泉徴収の記載要否は、取引内容や登録状況で変わるため、案件ごとに確認します。
