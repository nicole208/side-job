# 副業開始チェックリスト

## このメモの役割

副業を始める前後で、最低限先に決めておいた方がよいことを順番に確認するためのチェックリストです。会社員として副業を始める場合は、勤務先ルールや契約条件によって必要事項が変わるため、その確認を最優先にします。

## 開始前に確認すること

- [ ] 勤務先の就業規則で副業可否、申告要否、競業避止の有無を確認した
- [ ] 副業で受ける業務内容が、本業の機密保持や利益相反に抵触しないか確認した
- [ ] 報酬の受け取り方法を確認した
- [ ] 契約書や発注内容を、口頭だけでなく文章で確認できる状態にした
- [ ] 連絡用のメールアドレス、チャット、カレンダー運用を分けるか決めた

## 最初の1か月で整えること

- [ ] 入出金を追いやすくするために、口座や管理表の分け方を決めた
- [ ] 請求書の発行手順を決めた
- [ ] 証憑の保存先を決めた
- [ ] 経費記録の方法を決めた
- [ ] 月末に確認する項目を一覧化した

## 最低限そろえる管理項目

- 取引先名
- 案件名
- 契約開始日と終了日
- 報酬金額
- 請求日
- 支払期日
- 入金日
- 関連ファイルの保存先

## 最初に決めておくと楽なこと

### 連番ルール

請求書番号や案件番号のルールを先に決めておくと、後から検索しやすくなります。

例:

- 2026-001
- 2026-002

### 保存先ルール

案件ごとに「契約」「請求」「入金」「証憑」の単位で保存先を分けると、確定申告前の整理が軽くなります。

### 月次確認のタイミング

少なくとも月1回は、未請求、未入金、未整理の証憑がないか確認する日を決めておく方が安全です。

## 判断ポイント

- 副業の収入が単発中心なのか、継続案件中心なのか
- 経費が少ないのか、多いのか
- 取引先ごとに請求形式が違うのか
- 請求書発行、帳簿付け、申告をどこまで自分でやるのか
