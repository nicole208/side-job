# 副業開始後の最初の1か月でやること

## このメモの役割

副業開始直後は、案件対応そのものよりも、後で困らない管理の土台を作ることが重要です。このメモでは、最初の1か月で優先して整えるべき実務を順番に整理します。

## 1週目でやること

- [ ] 副業案件の一覧管理方法を決めた
- [ ] 契約、請求、入金、証憑の保存先を決めた
- [ ] 連絡手段と記録の残し方を決めた
- [ ] 本業と衝突しない作業時間帯を決めた

## 2週目までにやること

- [ ] 請求書番号や案件番号のルールを決めた
- [ ] 月次締めを実施する日を決めた
- [ ] 経費記録の方法を決めた
- [ ] 証憑を残す単位を決めた

## 3週目までにやること

- [ ] 売上と入金予定を一覧化した
- [ ] 未請求や未入金を確認する管理表を作った
- [ ] 私用混在支出の扱い方針を決めた
- [ ] 帳簿付けをどの方法で続けるか決めた

## 月末までにやること

- [ ] 初回の月次締めを試した
- [ ] 支出や売上の記録漏れがないか確認した
- [ ] 翌月に持ち越す課題を洗い出した
- [ ] 申告期に向けて不足している管理項目をメモした

## 最低限そろえたい管理表の列

### 案件管理

- 取引先名
- 案件名
- ステータス
- 契約日
- 納期
- 報酬額

### 請求管理

- 請求書番号
- 請求日
- 支払期日
- 入金予定日
- 実際の入金日

### 経費管理

- 支払日
- 支払先
- 内容
- 金額
- 仕事との関係
- 証憑保存先

## よくある後回しポイント

- 請求管理を頭の中だけで済ませる
- 証憑保存を後でまとめようとする
- 支出の用途メモを書かない
- 月次確認日を決めずに都度対応する
