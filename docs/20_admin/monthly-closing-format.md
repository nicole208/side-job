# 月次管理フォーマット

## このメモの役割

[../../templates/monthly-closing-checklist.md](../../templates/monthly-closing-checklist.md) をそのまま実運用に載せやすい形へ具体化した、月次締め用のフォーマットです。毎月この 1 ファイル形式で記録すれば、売上、請求、入金、経費、持ち越し課題をまとめて残せます。

## 使い方

1. 月末または翌月初に、このフォーマットを月ごとに複製する
2. チェック項目を埋めながら、表の数字と案件メモを更新する
3. 未完了事項は翌月への持ち越し欄に残す
4. 年末はこの月次記録を見返して不足資料を洗い出す

## 運用のコツ

- 数字だけでなく、止まっている理由を短く残す
- 金額不明なら空欄よりも確認中と書く
- 未請求や未入金は翌月に持ち越す前提で必ず残す
- この記録と請求管理表、経費管理表の数字がずれていないか確認する
