# 経費記録の基本

## このメモの役割

副業で発生した支出を、後から慌てずに整理できるようにするための運用メモです。重要なのは、節税テクニックより先に「何に使ったかを説明できる状態」にしておくことです。

## 支出が発生したらすぐ記録する項目

- 支払日
- 支払先
- 金額
- 内容
- 仕事との関係
- 支払方法
- 証憑の保存先

## 迷いやすい支出の整理観点

| 項目 | 見るポイント | メモ例 |
| ---- | ------------ | ------ |
| 通信費 | 私用と業務用が混ざるか | 使用割合を残す |
| 交通費 | 何の移動か説明できるか | 取引先訪問、打ち合わせ |
| 消耗品費 | 業務で使う物か | ケーブル、文具、小物 |
| ソフト利用料 | 業務に直接必要か | デザインツール、会計ソフト |
| 書籍、講座 | 仕事内容との関連が説明できるか | 受注業務に関連する学習 |

## 証憑の残し方

- レシートや領収書は撮影だけで終わらせず、何の支出か一言メモを付ける
- クレジットカード明細だけでなく、可能なら元の領収書も残す
- サブスクは契約画面、請求メール、利用明細のどれを残すか決めておく
- 電子データは月ごとにまとめる

## 私用と業務用が混ざる支出の考え方

- 全額ではなく、業務利用分だけを整理する前提で考える
- 割合の根拠を簡単に説明できるようにする
- 毎月ぶれるより、自分なりの基準を決めて継続する方が整理しやすい

## 月次チェックリスト

- [ ] その月の支出が管理表に入っている
- [ ] 証憑ファイルの保存漏れがない
- [ ] 内容が曖昧な支出に説明を追記した
- [ ] 私用混在の支出に自分の基準を書いた
- [ ] 年末にまとめ直さなくて済む程度に整理できている

## 判断ポイント

- 仕事との関係を第三者に説明できるか
- 一時的な思いつきではなく、継続して同じ基準で整理できるか
- 証憑だけでなく、用途メモも残っているか
