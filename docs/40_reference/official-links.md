# 公的情報リンク集

- 最終確認日: 2026-03-21

## このメモの役割

副業運用で参照頻度の高い公的情報の入口をまとめたリンク集です。ブックマークの代わりではなく、何を確認したいときにどこを見るかが分かる状態を目指します。

## 自治体関連

自治体ごとに内容が変わるため、次の観点で居住地の公式サイトを確認します。

- 住民税の案内
- 個人事業税の案内
- 国民健康保険の案内
- 申告書の提出先や期限

検索の目安:

- 自治体名 住民税 申告
- 自治体名 個人事業税
- 自治体名 国民健康保険 所得申告

## 使い方のルール

- ブログや SNS の解説を読む前に、まずここにある一次情報を確認する
- 制度変更の可能性がある内容は、最終確認日を自分のメモにも残す
- 迷った論点は URL だけでなく、見たページ名や確認日も記録する
