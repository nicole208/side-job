# 確定申告の全体像

- 対象年度: 2025年分の所得税確定申告を想定した整理
- 最終確認日: 2026-03-21
- 参照元:
  - 国税庁 確定申告特集: https://www.nta.go.jp/taxes/shiraberu/shinkoku/tokushu/index.htm
  - e-Tax: https://www.e-tax.nta.go.jp/
  - 国税庁 タックスアンサー: https://www.nta.go.jp/taxes/shiraberu/taxanswer/index2.htm

## このメモの役割

副業を始めた人が、年末から申告時期にかけて何をそろえればよいかを俯瞰するための入口メモです。個別論点の結論を断定するのではなく、何を確認し、どの順番で整理するかを中心にまとめています。

## まず押さえること

- 確定申告が必要かどうかは、収入の種類や金額、給与所得の有無などの条件で変わる
- 所得税の確定申告をしない場合でも、住民税の申告が必要になるケースがある
- 期限や必要書類は毎年の案内を必ず確認する

## 年間を通じて準備するもの

- 売上や報酬の記録
- 経費の記録と証憑
- 源泉徴収票
- 各種控除証明書
- 口座やカード明細の補助資料

## 申告前の流れ

1. 年間の売上と入金を締める
2. 経費と証憑を整理する
3. 控除に必要な書類を集める
4. 帳簿や集計表を見直す
5. 申告方法を決める
6. 提出前に入力内容と添付要否を確認する

## 迷いやすい判断ポイント

### 所得区分

副業収入が事業所得、雑所得などのどれに当たるかで整理の仕方が変わります。継続性、営利性、独立性などの事情で判断が分かれるため、安易に決め打ちしない方が安全です。

### 青色申告か白色申告か

どちらが適切かは、事前届出の有無、帳簿作成の体制、今後の継続性で判断が必要です。節税額だけではなく、継続して管理できるかも含めて考えます。

### 住民税の扱い

会社員の場合は、住民税の取扱いが本業側への見え方に関わることがあります。制度説明だけでなく、勤務先ルールや自治体案内も確認した方が安全です。

## 申告時期に確認すること

- その年の提出期限
- e-Tax 利用の事前準備の有無
- 添付や提出が必要な書類
- 定額減税など、その年特有の制度変更
- 申告後に納付が必要か、還付になるか

## ありがちな詰まりどころ

- 売上は分かるが、経費証憑が月ごとにそろっていない
- 仕事用と私用が混ざった支出の整理基準がない
- 源泉徴収された金額の扱いが分からない
- 控除証明書がどこにあるか分からない
- そもそも自分が所得税申告対象か住民税申告対象か曖昧

## 実務上の進め方

- 月次で売上と経費を締める
- 年末に一気にやらず、証明書類が届いたら保管場所を固定する
- 判断が難しい論点は、事実関係と参照元をメモに残す
- 最終的な提出前に、その年の国税庁案内を見直す

## 注意

このメモは一般的な整理用です。税務判断は個別事情で変わるため、最終的には国税庁や自治体の案内を確認し、必要に応じて税理士等への確認を検討します。
