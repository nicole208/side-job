# 少額減価償却資産の特例 2026年改正メモ

- 対象年度: 2026年4月1日以後に取得して事業に使用し始める資産を中心に整理
- 最終確認日: 2026-05-03
- 参照元:
  - 弥生株式会社「【2026最新】最大40万に！少額減価償却資産の特例と仕訳例｜改正対応」: https://www.yayoi-kk.co.jp/shinkoku/aoiroshinkoku/oyakudachi/shogakugenkashokyakushisan/
  - 財務省「令和8年度税制改正の大綱（3/9）」: https://www.mof.go.jp/tax_policy/tax_reform/outline/fy2026/08taikou_03.htm

## このメモの役割

2026年4月から、少額減価償却資産の特例で即時償却できる資産の取得価額上限が、従来の30万円未満から40万円未満に引き上げられる点を記録するためのメモです。

副業でパソコン、ソフトウェア、周辺機器、業務用備品などを買うときに、通常の減価償却にするか、少額減価償却資産の特例でその年にまとめて経費計上するかを判断する入口として使います。

## まず押さえること

- 2026年4月1日以後に取得して事業に使い始めた資産は、取得価額40万円未満が特例対象の目安になる
- 2026年3月31日までに取得した資産は、従来どおり取得価額30万円未満が目安になる
- 対象になるのは青色申告をしている中小事業者等
- 個人事業主の場合も、青色申告が前提になる
- 1年あたりの合計上限は300万円で据え置き
- この特例を使うか、通常どおり減価償却するかは選択できる

## 取得価額ごとの考え方

| 取得価額 | 主な扱い |
| --- | --- |
| 10万円未満 | 原則として減価償却せず、一時に必要経費にしやすい |
| 10万円以上20万円未満 | 一括償却資産として3年で均等償却する選択肢がある |
| 10万円以上40万円未満 | 2026年4月1日以後取得分は、青色申告なら少額減価償却資産の特例を検討できる |
| 40万円以上 | 原則として通常の減価償却を検討する |

## 変更点

### 取得価額の上限

少額減価償却資産の特例で即時償却できる取得価額の上限が、30万円未満から40万円未満に引き上げられます。

たとえば、2026年4月1日以後に35万円の業務用パソコンを取得して事業に使用し始めた場合、他の要件を満たせば、その年にまとめて必要経費にできる可能性があります。

### 適用期限

弥生の記事では、現在の対象は2029年3月31日までに取得して事業で使用し始めた資産とされています。

この特例は恒久制度ではなく、期限付きで延長や見直しが行われる制度なので、申告前には最新情報を確認します。

### 従業員数要件

弥生の記事では、個人事業主の場合、青色申告に加えて常時使用する従業員数が400人以下であることが要件として整理されています。

2026年3月31日までの取得分については500人以下とされていたため、取得価額の上限は広がる一方で、対象者側の要件は一部厳しくなっています。

## 適用するときの注意点

### 取得日と使用開始日を見る

40万円未満の基準は、2026年4月1日以後に取得して事業で使用し始める資産を前提に確認します。

3月末と4月初めをまたぐ購入は、請求日、納品日、検収日、使用開始日を後で説明できるように記録しておきます。

### 年間300万円の上限がある

少額減価償却資産の特例で一時に経費計上できる取得価額の合計は、1年あたり300万円までです。

複数の資産を取得する場合は、資産ごとの取得価額と、年間の累計額を管理します。

### 税込経理か税抜経理かで判定額が変わる

取得価額が40万円未満かどうかは、消費税の経理方式によって判定額が変わります。

- 税抜経理: 税抜金額で判定する
- 税込経理: 税込金額で判定する
- 免税事業者: 実務上、税込金額で判定する前提になりやすい

税込だと40万円以上になる買い物は、特例対象になるか注意して確認します。

### 事業に必要な資産であることが前提

趣味や私用のための資産は対象外です。仕事用と説明できる資産か、私用と混ざる場合はどのように按分・管理するかを記録します。

### 貸付用資産は注意する

2022年4月以後、主要な事業目的ではない貸付けに使うために取得した減価償却資産は対象外になる場合があります。

副業の本業と関係の薄い資産を節税目的で購入するような使い方は避けます。

## 申告時に残す情報

個人事業主でこの特例を使う場合は、青色申告決算書の「減価償却費の計算」欄への記載や、明細の保管が必要になります。

記録しておく項目:

- 取得日
- 使用開始日
- 資産名
- 取得価額
- 税込、税抜のどちらで判定したか
- 支払先
- 事業での用途
- 証憑の保存先
- 少額減価償却資産の特例を使うか、通常の減価償却にするか

青色申告決算書の摘要欄では、弥生の記事で「措法28の2」と明記する運用が紹介されています。
